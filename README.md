<h1>GPO Folder Share</h1>



<h2>Description</h2>
Step by step demonstration how to set up a Group Policy Object to access a shared folder.
<br />


<h2>Languages and Utilities Used</h2>

- <b>PowerShell</b> 
- <b>Diskpart</b>

<h2>Environments Used </h2>

- <b>Windows 10</b> (21H2)
